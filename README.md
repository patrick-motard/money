# money


Wealth & debt management tools.



## Install

```
git clone https://github.com/patrick-motard/money.git
cd money && npm i
```


## Features


### Daily Simple Interest Amortization

Example:
```
// from index.js
var p = 600; // principal
var m = 11; // first month - 1 (11 = december, 0 = january)
var y = 2012; // first year
var r = 0.2436; // interest rate
var smp = 173.86; // standar monthly payment
```
